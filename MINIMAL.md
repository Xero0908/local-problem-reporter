# ⚡ ABSOLUTE MINIMUM - Just Run It

You don't need the scripts. Just do this:

## Terminal 1 - Backend

```bash
cd backend
python run.py
```

That's it. Server runs on http://localhost:8000

## Terminal 2 - Frontend

```bash
cd frontend
npm start
```

That's it. App opens on http://localhost:3000
